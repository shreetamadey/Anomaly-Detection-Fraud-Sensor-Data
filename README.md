# Anomaly Detection for Fraud and Sensor Data
Comparative analysis of statistical and machine learning-based anomaly detection techniques for fraud detection using synthetic credit card transaction data.


**PROJECT OVERVIEW:**

This project was completed as part of the IDEAS Summer Internship Program 2026.

The objective is to investigate different anomaly detection techniques for identifying fraudulent transactions.


**LIBRARIES USED:**

NumPy

Pandas

SciPy

Scikit-Learn


**DATASET:**

A synthetic dataset consisting of:

950 normal transactions

50 fraudulent transactions


**FEATURES:**

Transaction Amount

Hour of Day

Transactions in Last 24 Hours

Distance from Home


**METHODS IMPLEMENTED:**

Z-Score Analysis

Interquartile Range (IQR)

Isolation Forest

Local Outlier Factor (LOF)


**RESULTS:**

Z-Score - Detects distance-based anomalies

IQR - Detects amount-based anomalies 

Isolation Forest - Best performance

LOF	- Lower accuracy
